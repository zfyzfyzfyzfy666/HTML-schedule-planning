# 📅 神秘人生大计

一个我自己在用的纯 HTML 日程管理工具。下载文件，浏览器打开，即开即用，数据存在本地，完全离线。

---

## ✨ 功能

- 📝 **新建 / 编辑事项** — 填写标题、截止日期、重要度（1～5 级）
- 🔁 **打卡循环事件** — 支持每 N 天循环打卡，可补打卡 / 撤销
- 🔢 **进度推进** — 点击「+1」逐步完成，进度满自动归档
- 🗂️ **筛选 & 排序** — 按待办 / 已完成筛选，按 DDL / 重要度等排序
- 📊 **统计面板** — 完成率、勤奋值折线图、重要度分布
- 🌙 **暗 / 亮主题** — 一键切换，默认湖蓝深色
- 💾 **导入 / 导出** — JSON 格式备份与恢复
