# 📅 神秘人生大计 · HTML 日程规划

> 一个纯前端、离线可用的个人日程管理工具，无需任何后端服务，双击 HTML 文件即可在浏览器中直接运行。

---

## ✨ 功能特性

| 功能 | 说明 |
|------|------|
| 📝 新建 / 编辑事项 | 通过弹窗填写标题、描述、截止日期、重要度 |
| 🔁 打卡循环事件 | 支持每 1～7 天循环打卡，可补打卡 / 撤销打卡 |
| 🔢 进度推进 | 点击「+1」逐步推进（共 6 步），进度满自动标记完成 |
| 🗂️ 筛选 & 排序 | 按全部 / 待办 / 已完成筛选；按 DDL、创建时间、标题、重要度排序 |
| 📊 统计面板 | 完成率、重要度分布、类型占比、勤奋值折线图 |
| 🌙 暗 / 亮主题 | 一键切换深色（湖蓝）和浅色两套主题 |
| 💾 数据导入 / 导出 | 支持 JSON 格式导出备份、导入还原 |
| 📴 完全离线 | 纯本地运行，数据保存在浏览器 localStorage，无需网络 |
| 🔔 音效反馈 | 通过 Web Audio API 动态合成点击音效，无需加载任何外部音频文件 |

## 🚀 快速上手

1. 将 `日程规划.html` 下载到本地。
2. 用任意现代浏览器（Chrome / Firefox / Edge / Safari）打开该文件。
3. 点击右上角「**新建事件**」按钮，填写信息后保存即可开始使用。

> **数据存储**：所有数据自动保存在浏览器的 `localStorage` 中，关闭页面后数据不会丢失。  
> 如需备份或迁移数据，使用右上角的「**⤓ 导出**」按钮导出 JSON 文件，在新设备上使用「**⤒ 导入**」按钮恢复。

## 🛠️ 技术说明

- **单文件纯 HTML**：CSS、JS 全部内联，无任何外部依赖，离线可用
- **数据层**：`localStorage`，键名为 `tasks`，格式为 JSON 数组
- **音效**：通过 Web Audio API 动态合成，无需加载外部音频资源
- **主题**：通过 CSS 自定义属性（`--变量名`）实现暗 / 亮双主题切换
- **最大进度步数**：默认 `MAX_PROGRESS = 6`，可在源码顶部修改
